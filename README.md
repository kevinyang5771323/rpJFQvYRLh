# 滑雪场管理系统 java871

## 项目概述

本项目是基于Spring Boot和Vue的滑雪场管理系统，适用于课程设计和毕业设计。

## 技术栈

- 后端：Spring Boot
- 前端：Vue

## 功能模块

### 系统角色

系统共有两类用户角色：管理员和普通用户。

#### 管理员

- 系统登录
- 管理员账户信息的数据维护
- 管理员条件检索查询
- 用户账户的数据维护
- 用户条件检索查询
- 滑雪设备的数据维护
- 滑雪设备条件检索查询
- 滑雪场地的数据维护
- 滑雪场地条件检索查询
- 滑雪教练的数据维护
- 滑雪教练条件检索查询
- 公告信息的数据维护
- 公告信息条件检索查询

#### 普通用户

- 系统登录
- 系统首页
- 系统公告查看
- 滑雪设备查看
- 滑雪设备租用
- 滑雪赛道查看
- 滑雪教练查看
- 滑雪教练预约
- 个人中心账户信息修改

## 系统运行环境

- 后端：Java环境（推荐使用JDK 1.8+）
- 前端：现代浏览器
- 数据库：MySQL（推荐版本 5.7+）
